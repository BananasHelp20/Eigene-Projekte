7) Settings
-----------

-neues Layout

-es wird ein neuer Button hinzugefügt (Pumpe hinzufügen), welcher .innerHTML +=

 Website der Code, der einzubauen ist ausgegeben. Dafür wird ein eigenes Div mit speziellem CSS
